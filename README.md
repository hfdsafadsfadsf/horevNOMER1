# Vizard Clone

Локальный аналог [Vizard.ai](https://vizard.ai/) на Python для личного пользования.

Кидаешь длинное видео → AI выбирает самые "вирусные" моменты → каждый автоматически кропается в 9:16 (Reels) с центрированием по лицу говорящего → накладываются премиум-субтитры с подсветкой текущего слова → опционально подмешивается твоя музыка.

## Что умеет

- ✔ Принимает **URL** (YouTube, TikTok, Vimeo, Twitch и 1000+ сайтов через yt-dlp) или **локальный файл**.
- ✔ **Транскрипция** через faster-whisper с word-level таймингами (поддерживает русский, английский и десятки других языков).
- ✔ **AI-выбор лучших моментов** через DeepSeek API (дёшево, ~$0.001 за видео).
- ✔ **Длина клипов на выбор:** 0–15 / 15–30 / 30–59 сек / Auto (максимум 59 секунд).
- ✔ **Smart Auto-Crop в 9:16** через OpenCV face detection — БЕЗ размытого фона, кадр центрируется по говорящему.
- ✔ **Библиотека из 9 премиум-шаблонов субтитров** (TikTok Classic, MrBeast, Hormozi, Minimal, Cinema, Pop Box, Karaoke Yellow, Neon Glow, Bebas Bottom) + полный кастом цветов/шрифтов/размеров/обводки/фона.
- ✔ **Субтитры исчезают в тишине** — никаких "висящих" слов когда человек молчит; пауза > 0.4с автоматически закрывает строку.
- ✔ **Karaoke выровнен по голосу** — подсветка слова идёт строго по таймингам whisper, не убегает вперёд.
- ✔ **Заголовки поверх клипа** — 6 готовых вариантов (Top Banner dark/red, Top Centered, Lower Third, Intro 3 sec, без заголовка). Текст можно задать вручную или взять из AI-предложения.
- ✔ **Логотип/обложка** — PNG/JPG в один из 6 углов, настраивается размер (% ширины), прозрачность, отступ.
- ✔ **Превью 15 секунд** — кнопка "Превью" в GUI генерирует короткий тест-клип со всеми твоими настройками за ~30 секунд. Можно итерировать стили БЕЗ перегона часового видео.
- ✔ **Пресеты** — все настройки (шаблоны, заголовок, логотип, музыка, длина) сохраняются в JSON-файл и загружаются одним кликом.
- ✔ **Музыка** — только твоя загруженная (никаких встроенных треков): одну на все клипы или отдельную под каждый. С sidechain compression (музыка автоматически приглушается на голосе).
- ✔ **Высокое качество:** 1080×1920, H.264 CRF 18, AAC 192 kbps, файлы 30–80 МБ за 60-секундный клип.
- ✔ **GUI** на CustomTkinter — никаких командных строк, кликабельный интерфейс.

## Установка на Windows (пошагово)

### Шаг 1. Python 3.10+

Скачать с [python.org](https://www.python.org/downloads/) (рекомендую 3.11 или 3.12).
**ВАЖНО:** при установке поставь галочку **"Add Python to PATH"**.

Проверь в командной строке:
```cmd
python --version
```
Должно вывести что-то вроде `Python 3.11.5`.

### Шаг 2. ffmpeg

Вариант А (через winget — самый простой, Windows 10/11):
```cmd
winget install Gyan.FFmpeg
```
Перезапусти cmd и проверь:
```cmd
ffmpeg -version
```

Вариант Б (вручную):
1. Скачай билд с https://www.gyan.dev/ffmpeg/builds/ → `ffmpeg-release-essentials.zip`
2. Распакуй, например в `C:\ffmpeg`
3. Добавь `C:\ffmpeg\bin` в системный PATH (Win + R → `sysdm.cpl` → Дополнительно → Переменные среды → Path → Изменить → Создать).
4. Перезапусти cmd и проверь `ffmpeg -version`.

### Шаг 3. Установка приложения

1. Скачай этот проект как ZIP (зелёная кнопка **Code → Download ZIP**) и распакуй, например в `C:\Users\YourName\vizard-clone\`.
   Либо клонируй через git:
   ```cmd
   git clone <repo-url>
   cd vizard-clone
   ```

2. Двойной клик по **`install_windows.bat`**.
   Скрипт создаст виртуальное окружение `.venv` и установит все зависимости. Это займёт 3–10 минут (faster-whisper и openai довольно большие).

3. Двойной клик по **`run.bat`** — откроется GUI.

### Шаг 4. DeepSeek API ключ

Если хочешь, чтобы AI выбирал лучшие моменты (рекомендую):

1. Зарегистрируйся на https://platform.deepseek.com/
2. Создай API key: https://platform.deepseek.com/api_keys
3. В GUI справа внизу в поле "DeepSeek API key" вставь ключ → "💾 Сохранить настройки".

Без ключа приложение всё равно работает — оно равномерно нарезает видео без анализа смысла.

## Использование (как пользоваться GUI)

1. **Слева в боковой панели**:
   - Вставь URL видео или нажми "📁 Выбрать файл с диска".
   - Выбери длину клипов: `0-15` / `15-30` / `30-59` / `auto`.
   - Задай минимум/максимум количества клипов (например, 3 минимум, 10 максимум).
   - Выбери язык видео или оставь `auto` для авто-детекции.
   - Настрой шрифт, цвета, размер субтитров.
   - Выбери режим музыки:
     - `none` — без музыки;
     - `common` — выбери один трек, он наложится на все клипы;
     - `per_clip` — выбери N треков, по одному на каждый клип (в нужном порядке).
   - Введи DeepSeek API key (опционально).
   - Выбери размер Whisper-модели (`small` — баланс качества и скорости; `large-v3` — максимум качества, но медленно).
   - Нажми "💾 Сохранить настройки" чтобы запомнить на будущее.

2. **Справа** нажми "🚀 Сгенерировать клипы".

3. В логе будет видно, что происходит: скачивание → транскрипция → AI-анализ → нарезка → кроп → субтитры → музыка. На 10-минутное видео обычно уходит 3–8 минут на современном CPU (без GPU).

4. Готовые клипы появятся в списке с кнопками "▶ Открыть" и "📂 В папке". Папка по умолчанию: `C:\Users\YourName\.vizard_clone\output\<имя_видео>\`.

## FAQ

**Q: Видео обрабатывается слишком долго.**
Самый медленный этап — Whisper-транскрипция. Если у тебя NVIDIA GPU:
1. Запусти `install_cuda.bat` (один раз) — установит CUDA-библиотеки (cuBLAS + cuDNN, ~700 МБ)
2. В GUI в секции "8. Качество видео" выбери "Устройство: auto" (или "cuda")
3. Whisper будет работать на GPU в 5–15 раз быстрее CPU

Если CUDA по какой-то причине не подцепилась — программа сама сделает fallback на CPU без падения.

Также можно поставить меньшую модель (`tiny`/`base`/`small`) — будет быстрее, но чуть менее точно.

**Q: Лицо иногда уходит за край кадра в 9:16.**
Это происходит на видео, где говорящий сильно двигается слева направо. Текущая версия рассчитывает ОДНУ среднюю позицию на весь клип (для стабильности). В планах — динамический tracking с плавным панорамированием.

**Q: Можно ли запустить без AI?**
Да. Сними галочку "Использовать AI для выбора моментов" — программа равномерно нарежет видео на N клипов по выбранной длине.

**Q: Можно ли поменять шрифт на свой?**
Да. Кинь свой `.ttf` файл в `resources/fonts/`, перезапусти приложение, выбери его в списке.

**Q: Поддерживается ли русский язык?**
Да, и транскрипция, и AI-анализ, и субтитры — всё работает на русском (faster-whisper и DeepSeek отлично понимают русский).
